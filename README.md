# 📚 Book Wellness

Book Wellness es una aplicación web tipo **Book Tracker** desarrollada con Flask. Su objetivo es ayudar a los usuarios a organizar sus lecturas, descubrir nuevos libros y en el futuro integrar un sistema inteligente de recomendaciones y bienestar lector.

⚠️ Este proyecto está actualmente en **desarrollo activo**.

---

## 🚀 Características actuales

- 🔎 Búsqueda de libros mediante API externa (Open Library)
- 📖 Visualización de detalles del libro (título, autor, portada, etc.)
- 📚 Estantería virtual de libros
- 🧭 Navegación entre páginas de búsqueda y detalle
- 🎨 Interfaz web con Flask + HTML + CSS

---

## 🧠 Visión del proyecto (IA + Book Wellness)

Este proyecto está evolucionando hacia un sistema más avanzado llamado **Book Wellness**, que integrará:

### 🤖 Agente de IA tipo “bibliotecario inteligente”
- Recomendación de libros basada en tus lecturas
- Sugerencias según estado de ánimo (mood reading)
- Análisis de hábitos de lectura
- Conversación con un asistente que entiende tus gustos

### 📊 Sistema de bienestar lector
- Estadísticas de lectura (progreso, constancia, géneros)
- Seguimiento de hábitos saludables de lectura
- Objetivos personales de lectura

### 🧩 Recomendaciones inteligentes
- Libros similares a los que has leído
- Recomendaciones personalizadas con IA
- Posible integración de modelos externos o APIs de IA
